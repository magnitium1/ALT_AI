🤖 AltAI - Структура проекта
================================

📁 Корневая папка: d:\Desktop\sait\AltAI\

📄 Основные файлы:
├── main.py              # Flask API сервер (порт 8070)
├── model.py             # Интеграция с AI моделью Qwen
├── requirements.txt     # Python зависимости
├── run.py              # Скрипт автоматического запуска
├── test_api.py         # Тестирование API
├── README.md           # Подробная документация
└── project_structure.txt # Этот файл

📁 Папки:
├── frontend/           # HTML фронтенд
│   └── index.html      # Веб-интерфейс с табами
├── venv/               # Виртуальное окружение Python
├── __pycache__/        # Кэш Python
└── .git/               # Git репозиторий

🚀 Как запустить:

1. БЫСТРЫЙ ЗАПУСК:
   python run.py

2. РУЧНОЙ ЗАПУСК:
   python main.py
   Открыть frontend/index.html в браузере

3. ТЕСТИРОВАНИЕ:
   python test_api.py

🌐 Доступ:
- Бэкенд: http://localhost:8070
- Фронтенд: frontend/index.html

📱 API эндпоинты:
- POST /register - регистрация
- POST /login - вход
- POST /request_to_model - AI модель (требует токен)
